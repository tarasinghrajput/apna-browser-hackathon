# Apna Browser

A lightweight experimental desktop browser built using Tauri.

Goal:
Build a minimal browser prototype in a 6-hour hackathon.

Core features:
- URL bar
- Back / Forward navigation
- Refresh
- Webpage rendering
- Custom homepage

Optional features:
- AI page summarizer
- Notes sidebar
- Website blocker

Constraints:
- Must remain lightweight
- Avoid heavy frameworks
- Prefer vanilla JS
