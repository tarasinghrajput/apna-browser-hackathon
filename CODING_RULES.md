# Coding Rules

The AI agent must follow these rules:

1. Keep the browser lightweight.

2. Avoid large frameworks like:
   - React
   - Angular
   - Vue

3. Use:
   - Vanilla HTML
   - Vanilla CSS
   - Vanilla JavaScript

4. Do not create unnecessary abstractions.

5. Implement features incrementally.

6. Always keep code readable for students.

7. Prefer simple solutions over complex architecture.

8. Do not add dependencies unless absolutely necessary.

9. The AI agent must never attempt to implement a browser engine.

10. The browser should only wrap the system WebView.
